## What is ILW?

ILW adds one clean geometric postulate to quantum mechanics: the space of decohered pointer states forms a Riemannian manifold with a hybrid Bures + configuration-space metric. This produces:
- Distance-dependent residual couplings that survive decoherence
- Geometric origin of the Born rule
- Curved Schrödinger equation on the manifold
- Semiclassical gravity sourced by web probability density
- Hybrid string-inspired compact dimensions
